# hxo-parser-stylus

Stylus parser for HXO framework

## 目的 (Purpose)

`hxo-parser-stylus` 用于处理 HXO 文件中带有 `lang="stylus"` 属性的样式块。

## 功能 (Features)

- **Stylus 支持**: 基础的 Stylus 语法解析。
